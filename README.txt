Doc's Alienation Weapon DPS Calculator by Dr C (drcpsn@hotmail.com)
Release 1.01 (12 Oct 2016)

A small app designed to optimize cores for weapons in the PS4 game Alienation.


1) Project Homepage:

https://alienationdps.github.io/alienationdps/


2) How to Use:

Windows: Run AlienationDPS-windows-xxx.exe installer, or extract
AlienationDPS-windows-xxx.zip to a folder and run AlienationDPS.exe.

Linux: Extract AlienationDPS-linux-xxx.tar.bz2 to a folder and run
AlienationDPS.


3) Building From Source:

AlienationDPS should build for Windows, Mac and most *nix platforms.

To build:
   1) Install Lazarus (http://www.lazarus-ide.org/)
   2) Extract AlienationDPS source archive
   3) Open project file (alienationdps.lpi) in Lazarus
   4) Hit Run -> Build
